**Maktavify** is a high-performance, 100% client-side engineering tool designed to parse, validate, visualize, and compare data structures with precision and speed.

Built with stability and data privacy in mind. No data leaves your browser.

![Project Status](https://img.shields.io/badge/status-stable-success.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## üöÄ Features

* **JSON Formatter & Validator:** Instantly process and beautify complex JSON data.
* **GraphQL Prettifier:** Standardize and format GraphQL queries/mutations.
* **Deep Diff Algorithm:** Hierarchically analyze differences between two datasets (Added/Removed/Modified).
* **Visualization Modes:** Tree View, Table View, and Raw Text.
* **Local Processing:** Zero API calls. All processing happens locally within the client.
* **Modern UI:** Fully responsive with Dark/Light mode support.

## üõ† Tech Stack

* **Core:** Next.js 15+ (App Router)
* **UI Engine:** Tailwind CSS
* **Animation:** Framer Motion & Canvas Confetti
* **Parser:** Prettier

## üì¶ Installation

Clone the repository:

```bash
git clone [https://github.com/contauto/maktavify.git](https://github.com/contauto/maktavify.git)
cd maktavify
````

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) with your browser.
